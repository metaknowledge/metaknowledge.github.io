HWS website

Last year (2024) I became president of Husky Winter Sports which is a ski club at the University of Washington. One of my main goals was to improve the website because the old one was hosting on wix which was not a great solution for the future. I decide to make the website as simple as possible to minimize the learning curve for future club members when they need to change the website. The website is pure html, and the whole site takes up about 7MB, including phots.

It has a few cool features like the "skip to main content" button is hidden until you tab. It also has a few animated images and the image carousel. All the interaction is made with css. Css has gotten really good in the past few years, and I see most websites not needing javascript for 90% of functionality in the future.

Training manual
